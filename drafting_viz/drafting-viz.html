Pre-Planning

Purpose:

Since FPM1, I have narrowed my questions and goals down to 3 main investigations. My goal is to understanding global CO2 and CH4 plume emissions temporally, spatially, and between sectors.

Carbon Mapper has 3 datasets: Sources, Sectors, and Plumes (for every year from 2016-2025). After investigating the data sets, Plume offers the most extensive data. Source and scenes data is limited and not contain the spatial and temporal extent of the plume yearly data. Therefore, to answer the questions, I will use the Plume data for each year and combine them with rbind. To look at plumes per country, I will join the Plume data set with global country data from spData::World.

# Create a function to plot total CO2 and CH4 concentrations overtime in specific countries 
country_time_plot <- function(country_name) {
  
  # Wrangle:
  plume_country_subset <- plumes_country %>% 
  filter(name_long == country_name) %>%  # Filter for Just China
  group_by(year = year(datetime), month = month(datetime), gas) %>%  # Find avg and sum for each month of each year 
  summarise(total_emissions = sum(emission_auto, na.rm = TRUE),
            avg_emissions = mean(emission_auto, na.rm = TRUE), 
            .groups = "drop") %>% 
  mutate(date = as.Date(paste(year, month, "01", sep = "-"))) %>% 
  st_drop_geometry()
  
  # CO2 plot overtime
  country_CO2 <- ggplot() + 
  geom_point(data = plume_country_subset %>% filter(gas == "CO2"), 
             mapping = aes(x = date, y = total_emissions), 
             color = gas_pal["CO2"]) + 
  geom_line(data = plume_country_subset %>% filter(gas == "CO2"), 
             mapping = aes(x = date, y = total_emissions), 
            color = gas_pal["CO2"]) + 
  theme_light() +
  labs(title = "Carbon Dioxide Plume Emissions", 
       x = "Time", 
       y = "Total Emissions (kg/hr)")
  
  # CH4 plot overtime
  country_CH4 <- ggplot() + 
  geom_point(data = plume_country_subset %>% filter(gas == "CH4"), 
             mapping = aes(x = date, y = total_emissions), 
             color = gas_pal["CH4"]) + 
  geom_line(data = plume_country_subset %>% filter(gas == "CH4"), 
             mapping = aes(x = date, y = total_emissions), 
            color = gas_pal["CH4"]) + 
  theme_light() +
    labs(title = "Methane Plume Emissions", 
       x = "Time", 
       y = "Total Emissions (kg/hr)")
  
  # Put China together
  (country_CO2 / country_CH4) + 
    plot_annotation(title = paste(country_name, "Plume Emissions Overtime"), 
                  subtitle = "IDK.", 
                  caption = "Data Source: Carbon Mapper",
                  theme = theme(plot.title = element_text(size = 17, face = "bold"), 
                                plot.subtitle = element_text(size = 12, color = "gray60"),
                                plot.caption = element_text(face = "italic")))
  
}
